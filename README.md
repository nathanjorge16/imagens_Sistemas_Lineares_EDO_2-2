# Estudo Dirigido de EDO - Análise de Sistemas Lineares

Este repositório contém os scripts em Python desenvolvidos para a análise gráfica do Trabalho de EDO (Tema 5: Foco Estável).

## Conteúdo
- `analise_grafica.py`: Script principal que gera o Retrato de Fases e a Evolução Temporal.
- `requirements.txt`: Lista de dependências (numpy, matplotlib, scipy).

## Como executar
1. Instale as bibliotecas: `pip install -r requirements.txt`
2. Execute o script: `python analise_grafica.py`
